## iThumb
Typecho文章缩略图(Post Thumbnail)插件
基于[SlantedExtend](https://github.com/DT27/SlantedExtend "SlantedExtend")的`thumbUrl`自定义字段增强增加了图片选择框。图片选择是从当前文章附件列表选取的，目前不支持从上传的所有文件中选取
### Usage
插件目录必须为iThumb
前台通过自定义字段获取缩略图链接
```php
<?php $this->fields->thumbUrl; ?>
```
